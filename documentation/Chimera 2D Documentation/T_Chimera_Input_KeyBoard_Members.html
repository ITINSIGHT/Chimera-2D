KeyBoard Members

 [This is preliminary documentation and subject to change.]

This Class Allow You To Manages the changing state of The Keyboard. Be sure to call Update() every frame before accessing the data. This is a static class (Module in Visual Basic)static classModule and so cannot be inherited or instantiated.


The following tables list the members exposed by KeyBoard.
