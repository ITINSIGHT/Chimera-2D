Chimera Games Engine Library Documentation
Chimera.GUI.WindowSystem Namespace
See Also Example Hierarchy

Classes

  Class Description
Bar A graphical bar drawn with two end sprites, and a stretched middle sprite.
Box A graphical box drawn with four corner sprites, and five other sprites stretched to allow dynamic resizing.
CheckBox A graphical checkbox button, with a description label.
ComboBox A graphical combobox control.
ComponentSkin Represents a GUI skin, with a reference to the source texture, and a list of GUIRect objects which make up the different parts of the skin.
DefaultSingleSkin Class Allows for default control skins to be accessed, and can be used to populate skins in a SkinnedComponent.
DefaultSixSkins Class Allows for default control skins to be accessed, and can be used to populate skins in a SkinnedComponent.
DefaultThreeSkins Class Allows for default control skins to be accessed, and can be used to populate skins in a SkinnedComponent.
Dialog A dialog box is a window that should return a value, defined by the DialogResult enumeration.
DisplaySettingsDialog A dialog box that allows the user to set the display resolution, and choose between fullscreen and windowed modes.
DrawableUIComponent UIComponent that draws itself and children to a texture, then draws that texture onto the screen. Drawing to a texture means the GUI to limit number of draws, and can allow cool effects like transparency and animations simpler.
GUIManager This is a game component that implements IUpdateable. It manages GUI components, in particular passing on events and handling focus.
GUIRect Combines two Rectangles representing the source and destination of an image.
Icon A static image from the GUI texture. Can be scaled to fit the control size.
Image A static image.
ImageButton A button comprised of seperate images for each state.
Label A graphical text label.
ListBox A graphical listbox control. Uses a scrollbar to allow more entries than can fit on control. Entries are actually children to a DrawableUIComponent, which acts as a viewport, only allowing some to be seen at one time. Scrollbar is only shown if required.
MenuBar Graphical bar for dropdown menus. Automatically stretches to the width of the parent control.
MenuItem A graphical menu item, that can be clicked, or contain child menu items that are shown as a popup menu.
MessageBox A popup message box, that displays a message to the user, and waits for a reply. Message boxes can contain an OK button, or an OK and Cancel button.
MovableArea A non-graphical control that moves it's parent control when dragged with the mouse.
PopUpMenu A graphical popup menu used by menu bars, and as context menus.
RadioButton RadioButton is exactly the same as the CheckBox control, except for different default graphics.
RadioGroup Contains multiple RadioButton controls, and handles the exclusively checked behavior. A radio button on it's own is simply a checkbox with a different default appearence.
ResizableArea A non-graphical control that resizes it's parent control when dragged with the mouse.
ScrollBar A graphical vertical scrollbar. To use just set the maximum value, and register for the OnScroll event.
SkinnedComponent Control with one or more skin states, allowing any component with a graphical skin to inherit the functionality of This Class.
TextBox Graphical textbox control. Very simple, only writes characters and deletes with the backspace. May be Updated in the future.
TextButton A push button comprised of a graphical Bar component, with a text label on top.
UIComponent This is the base public class for all GUI controls. Handles all the shared events.
Window Represents a graphical window, with a title bar, a close button, and movable and resizable areas. Most aspects can be removed, and the movable area can cover the whole window, like WinAmp or iTunes.

Interfaces

  Interface Description
ISingleSkin Interface allows for public classes containing SkinnedComponent objects, to pass interfaces for changing skin locations, rather than the full object.
ISixSkins  
IThreeSkins Interface allows for public classes containing SkinnedComponent objects, to pass interfaces for changing skin locations, rather than the full object. This version has the regular state from ISingleSkin, as well as states for hover and pressed states.

Delegates

  Delegate Description
ClickHandler When a control has been clicked (mouse pressed then released inside the area of a control).
CloseAllHandler When all menus should be closed.
CloseHandler When a control such as a window is closed.
EndMovingHandler When a control finishes moving.
EndResizingHandler When a control finishes resizing.
GetFocusHandler When a control receives focus.
LoseFocusHandler When a control no longer has focus.
MouseOutHandler When the mouse leaves a control's location rectangle.
MouseOverHandler When the mouse enters a control's location rectangle.
MoveHandler When a control is moved.
PopUpClosedHandler When a popup menu is closed.
PopUpOpenHandler When a popup menu is opened.
RequiresRedrawHandler When a control requires a redraw.
ResizeHandler When a control is resized.
ScrollHandler When scrollbar's position changes.
SelectionChangedHandler When a listbox selection changes.
StartMovingHandler When control begins moving.
StartResizingHandler When a control begins resizing.

Enumerations

  Enumeration Description
DialogResult Result of a dialog or message box.
MessageBoxButtons Type of buttons to appear on message box.
MessageBoxType The type type of information being displayed. An icon representing type is shown.
MouseState Possible mouse cursors.
ResizeAreas Possible resizable area types.
SkinState Handy for referring to skins by name rather than by their index.
