Body

Body is at the core of the Rigid Body System. It is a RigidBody without the geometry.

The Body handles all the physics of motion: position, velocity, acceleration forces, torques, etc...

  1. -Set forces and torques (gravity, springs, user input...)
  2. -Apply forces and torques (updating velocity only)
  3. -Update positions and orientations

In technical terms, this is known as Symplectic Euler Integration because velocity is Updated prior to position (The reverse of simple Euler Integration)

PolygonRigidBody Represents a polygon riged body and acts as base public class for other polygon primatives: rectangle, circle, point.

By default, this object will use the "Relative" collison precision type. CollisionPrecisionType

To manually construct this object, set all the properties then call SetGeometry and SetGrid in succession to initialize the collison capability.

CollisionPrecisionType Determines how the collision grid is constructed.

If 'Relative', the collision precision will be multiplied by the length of the shortest edge of the geometries AABB. This value will then be used for the grid's cell size

If 'Absolute', the collision precision value will be used directly for the grid cell size.
